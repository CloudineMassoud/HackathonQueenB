# 🌸 BeSafe - Your Digital Big Sister

**BeSafe** is an empowering AI-powered support platform developed for the **QueenB BeSafe Hackathon 2026**. Built with the MERN stack and enhanced with RAG (Retrieval-Augmented Generation), BeSafe provides a safe, supportive space for conversations about body positivity, relationships, intimacy, and nutrition.

---

## ✨ Features

### 🤖 AI-Powered Chat Support
- Conversational AI assistant ("Big Sis") powered by OpenAI
- Context-aware responses using RAG with vector embeddings
- Multi-language support (Hebrew & English)
- Quick topic buttons for easy conversation starters

### 📚 Content Library
Educational resources organized by categories:
- 💪 **Body Positivity** - Self-love and body acceptance
- 💕 **Relationships** - Healthy relationship guidance
- 🌹 **Intimacy** - Safe and informed discussions
- 🥗 **Nutrition** - Balanced eating and wellness

### 🎨 Customizable Experience
- Three beautiful themes: Pink (default), Mint, and Dark
- Responsive design for all devices
- Smooth, intuitive user interface

### 🧠 RAG Knowledge Base
- Vector-based document retrieval using Supabase
- Tag-based organization for precise content matching
- Easy-to-extend knowledge base system

---

## 🛠️ Tech Stack

### Frontend
- **React** (v19) with **Vite** for fast development
- **React Router** for navigation
- **Axios** for API calls
- **CSS Modules** with custom theming

### Backend
- **Node.js** with **Express** (v5)
- **OpenAI API** for chat and embeddings
- **Supabase** for vector database and storage
- **CORS** configured for secure cross-origin requests

### AI & Data
- OpenAI `text-embedding-3-small` for vector embeddings
- Supabase `pgvector` for similarity search
- RAG pipeline for context-enhanced responses
